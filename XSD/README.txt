This is a simple XML against XSD Validator for Python 2.7 - 3.2
Eventually it could work with other versions too.

The extended description can be found at: http://hahamo.wordpress.com/2013/05/09/validating-xml-against-xsd/

To run the Script you have to install lxml (http://lxml.de)

Usage:
python XSDValidator.py <list of XML files> <list of XSD files>

For the Windows distributable version the usage:
XSDValidator.exe <list of XML files> <list of XSD files>
